# Why This Project Exists

このプロジェクトは、「古いサブMacを横に置いて使いたいが、キーボードの接続切替が面倒」というかなり限定された問題を解くために始まりました。

## Apple純正機能では足りなかった理由

Apple 純正の Universal Control は完成度が高い一方で、対応する macOS バージョンとハードウェアに制約があります。古い Intel Mac や Big Sur 環境では対象外になることがあり、その場合は採用できません。

## 既存OSSではなく専用実装にした理由

Barrier や Input Leap は多機能で便利ですが、そのぶん次のような難しさがあります。

- macOS 権限まわりの調整が重い
- 古い macOS で挙動差が出やすい
- 修飾キーや配列差で違和感が出やすい
- 今回ほしい機能に対して守備範囲が広すぎる

## このプロジェクトの割り切り

代わりに、要件を次のように絞りました。

- キーボードだけ
- LAN 内だけ
- 片方向だけ
- Mac から Mac だけ

この割り切りにより、実装量とトラブルの種類を減らし、古い環境でも扱いやすい最小構成を目指しています。

## つまり何が価値か

このプロジェクトの価値は「汎用入力共有ツール」であることではなく、次の条件が揃う人にとって実用的な最小解になることです。

- 古い Mac をサブ機としてまだ活かしたい
- Apple 純正機能の対象外である
- 既存の高機能OSSより、用途限定で単純なものがほしい
- 自分でコードを追える構成のほうが安心できる
