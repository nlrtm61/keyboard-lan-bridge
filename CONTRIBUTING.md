# Contributing

Issue と Pull Request は歓迎します。

## まず確認してほしいこと

- このプロジェクトは LAN 内のキーボード転送に用途を絞っています
- マウス共有、クリップボード共有、リモートデスクトップ化は現時点の主目的ではありません
- 古い macOS 環境でも成立する設計を優先しています

## バグ報告

次の情報があると助かります。

- macOS バージョン
- 送信側/受信側の役割
- JIS / US などのキーボード配列
- 再現手順
- `sender` / `receiver` のログ

公開Issueには、実IP、ホスト名、個人パス、共有トークンを書かないでください。

## Pull Request

- 変更理由を README または docs に反映してください
- 権限要件や Big Sur 互換性に影響する場合は明記してください
- ローカル設定ファイルやビルド生成物は含めないでください
